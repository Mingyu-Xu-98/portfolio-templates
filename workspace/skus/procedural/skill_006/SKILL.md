# 知识问答挑战流程

## 概述
用户回答环境相关知识问题，获得XP奖励并完成相关任务。

## 流程步骤

1. **初始化问答**
   - 加载问答数据
   - 显示进度指示器（圆点序列）
   - 初始化分数为0

2. **题目展示**
   - 显示当前题号和总题数
   - 显示问题文本
   - 显示四个选项按钮

3. **选择答案**
   - 点击选项
   - 立即锁定选择（不可更改）
   - 更新分数（正确则+1）

4. **结果反馈**
   - 正确选项：绿色边框 + 勾选图标
   - 错误选择：红色边框 + 叉号图标
   - 显示解释文本

5. **导航控制**
   - 显示"下一题"或"查看结果"按钮
   - 最后一题完成后显示最终得分

6. **完成处理**
   - 记录问答完成状态和分数
   - 发放100 XP
   - 自动完成关联任务
   - 返回上一页面

## 状态管理
```
currentQ: 当前题目索引
score: 正确答题数
selected: 当前选择
answered: 是否已作答
```

## 视觉反馈
- 未作答：默认边框颜色
- 已选择未提交：主题色边框
- 正确：success色边框和背景
- 错误：error色边框和背景

## 预期结果
- 问答记录保存到completedQuizzes
- XP即时发放
- 关联任务自动标记完成
