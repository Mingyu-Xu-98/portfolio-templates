# 谈判决策与持续改进

## 何时不应进行谈判

### 评估标准
1. **谈判成本大于效益**：当时间就是金钱时，微小的利益不值得耗费大量时间
2. **BATNA极差且众所周知**：此时没有筹码，谈判可能招致反感
3. **谈判会发出错误信号**：可能让你看起来贪婪或不信任对方
4. **谈判对关系的潜在伤害大于价值**：过度压价可能破坏长期合作关系
5. **谈判是文化禁忌**：在特定文化背景下，某些场景议价被视为无礼
6. **你的BATNA优于对方提议**：直接离开，启动备选方案

## 谈判能力的持续提升

### 区分归因方式
- **外归因（无效）：** 怪罪市场、时间、对手条件等不可控因素
- **内归因（推荐）：** 反思准备不足、信息缺失、策略失误等可控因素

### 循序渐进地应用策略
1. 列出想尝试的策略
2. 每周挑选一个策略深入思考
3. 找"魔鬼代言人"讨论和质疑
4. 在实际谈判前制定详细计划
