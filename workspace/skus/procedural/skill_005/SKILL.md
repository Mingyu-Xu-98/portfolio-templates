# 物种收集工作流

## 概述
用户通过地图探索发现环境中的物种，可查看详情并收集到个人图鉴中。

## 发现流程

1. **地图探索**
   - 在交互地图上点击环境标记点
   - 显示环境信息卡片
   - 点击"探索"按钮打开发现模态框

2. **物种列表展示**
   - 模态框显示该环境所有物种
   - 每个物种卡片包含：
     - 缩略图
     - 中英文名称
     - 描述（2行预览）
     - 稀有度标签
     - 收集状态

3. **查看详情**
   - 点击物种卡片
   - 导航到SpeciesDetail页面
   - 自动标记为已阅读
   - 自动收集到图鉴（如未收集）

4. **收集操作**
   - 未收集物种显示"采集"按钮
   - 点击采集：
     - 添加到collectedCards
     - 获得25 XP
     - 解锁相关知识卡牌
   - 已收集物种显示"已采集"状态

## 稀有度显示
```
legendary（传说）：金色标签
rare（稀有）：蓝色标签
common（普通）：绿色标签
```

## 状态同步
- collectedCards：已收集物种ID列表
- readSpecies：已阅读物种ID列表
- unlockedCards：已解锁知识卡牌ID列表

## 预期结果
- 物种添加到个人收藏
- 相关知识卡牌自动解锁
- 收藏进度百分比更新
