# 虚拟探险完成流程

## 概述
用户通过逐步完成探险步骤、解答谜题来探索中国极致环境，最终获得洞察和奖励。

## 步骤

1. **初始化探险**
   - 加载探险数据，显示进度条（每步一个圆点）
   - 显示当前步骤编号和总步骤数

2. **阅读步骤内容**
   - 阅读步骤标题和叙述文本
   - 查看关键事实列表
   - 如有物种聚焦，可点击查看物种详情

3. **处理谜题（如有）**
   - 若步骤包含谜题，点击"挑战谜题"按钮
   - 根据谜题类型作答：
     - **quiz（知识问答）**：选择正确选项
     - **identify（物种识别）**：根据描述选择对应物种
     - **reasoning（环境推理）**：选择合理解释
     - **ordering（排序挑战）**：按正确顺序排列选项
   - 可查看提示（hint）获取帮助
   - 提交答案后查看解释
   - 正确则获得30 XP和解锁奖励卡牌

4. **导航控制**
   - 完成谜题后点击"继续"进入下一步
   - 可点击"上一步"返回查看之前内容

5. **完成探险**
   - 最后一步完成后，显示探险洞察
   - 获得150 XP和纪念品
   - 探险标记为已完成

## 决策点
- 是否有谜题？无则直接继续
- 谜题是否已解答？未解答则需先完成
- 是否为最后一步？是则触发完成流程

## 预期结果
- 探险状态更新为已完成
- 用户获得XP和纪念品奖励
- 相关知识卡牌解锁
