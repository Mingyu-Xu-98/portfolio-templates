# 知识卡牌解锁系统

## 概述
知识卡牌通过多种用户行为触发解锁，包括访问环境、阅读物种、完成谜题等。

## 解锁触发类型

### 1. 按环境解锁
```
触发条件：访问特定环境页面
解锁范围：该环境的地理类卡牌
实现方法：unlockByEnvironment(environmentId)
```

### 2. 按物种解锁
```
触发条件：阅读物种详情或收集物种卡牌
解锁范围：该物种相关的所有卡牌
实现方法：unlockBySpecies(speciesId)
```

### 3. 按谜题解锁
```
触发条件：正确解答探险谜题
解锁范围：谜题指定的奖励卡牌
实现方法：unlockByPuzzle(cardId)
```

### 4. 直接解锁
```
触发条件：特定事件完成
解锁范围：指定卡牌ID
实现方法：unlockCard(cardId)
```

## 卡牌分类
- **geography（地理）**：地形、位置、地质信息
- **nature（自然）**：气候、生态过程
- **species（物种）**：动植物信息
- **history（历史）**：人文历史
- **culture（文化）**：当地文化

## 稀有度等级
- **common（普通）**：基础信息，易解锁
- **rare（稀有）**：深入内容，需特定行为
- **legendary（传说）**：珍稀内容，需完成挑战

## 预期结果
- 解锁的卡牌ID添加到unlockedCards列表
- 用户可在图鉴页面查看已解锁卡牌
- 进度百分比实时更新
